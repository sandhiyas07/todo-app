# todo-app
A simple To-Do App built using HTML, CSS, and JavaScript to manage daily tasks with add, delete, and complete features.

## 📌 Description
A simple and interactive To-Do App built using HTML, CSS, and JavaScript. This app helps users manage daily tasks by allowing them to add, complete, and delete tasks. The data is stored in the browser using localStorage.

---

## 🚀 Features
- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Save tasks using localStorage
- 📱 Clean and user-friendly interface

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
